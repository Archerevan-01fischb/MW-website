MCP MANUAL SEARCH

An MCP server that provides search capabilities for the Midwinter game manual database.

WHAT IS THIS?

The Midwinter Database Search MCP Server allows Claude (or other MCP-compatible AI assistants) to search through comprehensive documentation about the original 1989 Midwinter game.

Query character stats, game mechanics, locations, items, and lore directly from your AI assistant.

FEATURES

  • ✓ Full-text search across game manual content
  • ✓ Character attribute lookups
  • ✓ Location and building information
  • ✓ Game mechanics documentation
  • ✓ SQLite-based for fast queries
  • ✓ No external dependencies

CONFIGURATION

Add to your .mcp.json in the project root:

{
  "mcpServers": {
    "midwinter": {
      "command": "python",
      "args": ["mcp-midwinter-search/server.py"]
    }
  }
}

REQUIREMENTS

• Python 3.10+
• pip install mcp
• Database file at database/midwinter_unified.db (included in repo)
